# Fountain Onboarding Website

An internal onboarding portal for new employees joining Fountain, a concierge telemedicine platform. This website provides new hires with essential resources, training materials, and information to help them get started successfully.

## Overview

This website serves as a comprehensive onboarding board for new Fountain employees, guiding them through their first days and weeks with the company. The design follows Fountain's branding guidelines with a clean, modern healthcare aesthetic while focusing on employee resources and company information.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design matching Fountain's brand identity
- **Smooth Animations**: Subtle scroll animations and hover effects for enhanced user experience
- **Navigation**: Sticky navigation bar with smooth scrolling to sections
- **Contact Form**: Interactive contact form with validation for employee questions
- **Onboarding Steps**: Clear 4-step onboarding process guide
- **Resources Section**: Access to employee handbook, tools, and training materials
- **Team Information**: Key contacts and team members to help new hires
- **Company Benefits**: Overview of what makes Fountain a great place to work

## Sections

1. **Navigation Bar**: Sticky header with logo and navigation menu
2. **Hero Section**: Welcome message for new employees with key onboarding timeline
3. **Getting Started**: 4-step onboarding journey (First Day, Company Overview, Role Training, Meet the Team)
4. **Resources**: Employee handbook, tools & systems, and learning & development resources
5. **Why Fountain?**: 6 key benefits of working at Fountain
6. **Meet Your Team**: Key contacts including manager, HR, and onboarding buddy
7. **Important Dates**: 30-day onboarding timeline and milestones
8. **Help & Support**: Contact form and support resources
9. **Footer**: Quick links and company information

## Getting Started

1. Open `index.html` in a web browser
2. No build process or dependencies required - pure HTML, CSS, and JavaScript
3. For development, use a local server (optional):
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

## Customization

### Colors
The color scheme is defined in CSS variables at the top of `styles.css`:
- Primary Color: `--primary-color`
- Secondary Color: `--secondary-color`
- Accent Color: `--accent-color`

### Typography
The website uses the Inter font family from Google Fonts for a modern, clean look.

### Content
All content can be edited directly in `index.html`. The structure is semantic and easy to modify.
